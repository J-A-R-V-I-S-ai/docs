Installing Using Git

1. Use the link provided to access the Git repository on Github.

2. Using your method of choice (terminal-based or Github-based), clone the repository to your computer.

3. Extract the repository on your computer, then open up the directory in the terminal

4. Locate the directory in JARVIS has been downloaded using the terminal. Then using the terminal, run the main.py python file using the command python3 main.py
