Commands

Open Commands

To open an app, simply say open app_name

Close Commands

To close an app, simply say close app_name

Weather Commands

To get the weather, simply say what is the weather in location?

Search Commands

To do an internet search, simply say search speech recognition using broswer

To go to a url, simply say go to url.com using browser

Math Commands

To add numbers, simply say what is number plus number

To subtract numbers, simply say what is number minus number

To multiply numbers, simply say what is number times number

To divide numbers, simply say what is number divided by number

To square numbers, simply say what is number to the power of number

Timetable Commands

To add an event to the timetable, simply say add event_name with a due date of 2020/25/4

To remove an event, simply say remove event_name from events

To check how many days till an event, simply say check when event_name is due

To clear all events, simply say clear all events

To get all events, simply say get events

Map and Location Commands

To search something using Google Maps, simply say search restaurants near me using google maps

To locate an address, simply say locate address

Time Commands

To ask the time, simply say what is the time?

To ask the time in an area, simply say what is the time in area_name?

Wake Detection

To wake JARVIS, simply say Hey JARVIS

Conversational Commands

To ask its name, simply say what is your name?

To ask how it is doing, simply say how are you?
