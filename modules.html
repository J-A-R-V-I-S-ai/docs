Modules

App Manager

Manages the opening and closing of apps

Audio Manager

Manages all volume related commands (muting, turning the volume up, down, etc.)

Calculator

Built-in calculator for JARVIS that calculates basic functions like adding, subtracting, multiplying, dividing and computation of powers (indices)

Command Handler

The Command Handler is the module that handles all your commands, and sending requests to other modules like App Manager to do a task for you; an example is opening microsoft word on your laptop.

Map Manager

Manages location and map related commands. Check the commands docs to see what commands are currently available.

Time Manager

Manages all time-related commands. This includes getting the time, getting the time in a particular city, etc.

Timetabler

JARVIS has an built-in timetabler, which schedules tasks and events for you. The timetabler module manages all commands related to scheduling tasks and events. This can range from get all the events and their corresponding due dates, to automatically being notified on when an event is only a few days away.

Weather Manager

Performs a variety of weather-related tasks like getting the weather today, tomorrow, etc.

Web Searches Manager

Manages all search-related commands, making sure to search on your browser of choice.
